# Skill & Plugin Quick Reference

Proactively suggest these when the task matches. Damon has many tools installed and cannot remember them all.

## Session & Workflow
| Command | When to suggest |
|---------|----------------|
| `/plan` | Interview-style planning session. Auto-scouts plugins at Step 1.5 |
| `/sync-context` | End of every session. Notes, session log, memory check, Productive queue |
| `/dispatch` | Kick off background work. Always include Write permission |
| `/jobs` | Check status of dispatched background tasks |
| `/bat-cave` | Open Command Center hub (ei-command-center-static.vercel.app) |
| `/plan-dashboard` | Open plans dashboard directly |
| `/compact-check` | Check context usage before it fills, good before long tasks |
| `/scope` | Clarify task scope before starting work |
| `/check-cost` | Estimate token cost for a planned operation |

## Review & Quality
| Command | When to suggest |
|---------|----------------|
| `/plan-review` | After writing any plan. Adversarial review, 4 modes (expand/selective/hold/reduce) |
| `/devils-advocate` | Before opening a PR. Structured code review debate |
| `/gemini-code-review` | Send code to Gemini for senior review |
| `/pr-pln-sct-plgns` | Before `/plan` or starting new work. Scouts which plugins would help |
